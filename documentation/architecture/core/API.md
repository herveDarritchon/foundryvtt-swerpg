# Public API - Interface de Développement

## 🎯 Vue d'ensemble

Documentation de l'API publique swerpg.api exposée pour les modules et extensions.

## 🏗️ Structure de l'API

```javascript
swerpg.api = {
  applications, // Classes d'applications
  canvas, // Composants canvas
  dice, // Système de dés
  documents, // Classes de documents
  models, // Modèles de données
  methods, // Méthodes utilitaires
  talents, // Système de talents
  hooks, // Gestionnaires de hooks
}
```
